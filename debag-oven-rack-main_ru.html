Принадлежности

Тележечные печи

Наши тележечные хлебопекарные печи позволяют экономить место и

энергию. Это возможно благодаря вертикальным тележкам и технике

MONSUN. Воздух обдувает изделие из теста со всех сторон при низкой

скорости. Отсутствие слепых зон при обдуве позволяет плотнее укладывать

изделия на противни. Таким образом вы сможете выпечь больше за то же

время.

Обзор продукции

MONSUN SM

Идеальное сочетание подовой и тележечной печи.

MONSUN S4/TWIN

Две тележки на каждый процесс выпекания — для увеличения количества выпечки вдвое за то же самое время.

MONSUN S5

Идеальный вариант для выпекания больших объемов — партия за партией.

Комбинированная печь

MONSUN SM

Сочетание подовой и тележечной печи открывает множество возможностей применения в пекарне. Этот модуль (î желанию оснащаемый каменной плитой) позволяет выпекать максимальное количество изделий - при этом для установки печи необходимо лишь 3,4 м2. В качестве альтернативы эти печи позволяют использовать тележки для выпекания.

Выверенная технология MONSUN обеспечивает превосходное качество выпечки, которая долго сохраняеес.
